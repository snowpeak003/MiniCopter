使用6050模块时，姿态解算使用互补滤波，Kp和Ki可以根据情况调整；
使用串口连接M3C模块时，M3C姿态解算使用卡尔曼；
遥控器使用串口6，也可以使用PWM的接收机或者PPM的接收机。硬件上支持SBUS，不过没实际测试过。
目前只是用了一个简单的外环PID，预先移植了串级PID，反注释即可；
上位机使用mwc上位机，可以读出和写入PID参数；
也可以使用printf函数用arduino的串口绘图或者Serialchart绘图,便于临时参数的调整；
写好了与上位机使用SPI通信的接口，实测可用，可以与Nano Pi,树莓派等连接，跑一些比较高级的应用。
